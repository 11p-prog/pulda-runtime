# 실행 인계서

## 사용자가 할 일

1. 압축 해제
2. Python 3.12 또는 Docker 설치 확인
3. `.env.example`을 `.env`로 복사
4. 실행
5. 브라우저에서 Event 입력
6. Notion 자동동기화를 원하면 Integration Token과 대상 Page ID 입력
7. GitHub 자동동기화를 원하면 PAT와 저장소명 입력

## Replit/Codex가 할 일

- Replit: 저장소 Import, Secrets 등록, Always On 배포
- Codex: UI 개선, LLM 분류기 연결, 테스트 확장, Connector 추가
- GitHub Actions: 테스트 및 배포 자동화

## ChatGPT가 할 일

- Theory/Architecture/Domain 정책 검토
- 사용 로그 리뷰
- 분류 규칙과 안전장치 개선 제안
- Notion 지식 저장소 문서 업데이트
- 릴리스 검토

## 현재 막힌 것

현재 연결된 GitHub App에는 접근 가능한 저장소가 0개이므로 이 세션에서 직접 Push하지 못했다.
