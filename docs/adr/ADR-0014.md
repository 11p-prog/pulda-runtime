# ADR-0014 — AI Control Tower Portability

Status: Approved  
Decision date: 2026-07-15  
Canonical authority: Notion `06 Decision Log`, ADR-014

## Context

Pulda OS currently uses ChatGPT as a primary control-tower assistant, but its
identity, operating memory, judgment rules, permissions, and evidence must not
belong to ChatGPT or to any single model provider. Provider policy, model
availability, sovereign-AI requirements, cost, capability, or privacy
constraints may require another model to assume some or all AI roles.

Decisions that exist only in project memory or chat history create a hidden
provider dependency and cannot be reproduced or inherited by another Builder.

## Decision

The human remains the final authority. The Pulda Control Plane preserves the
authoritative sources, operating loop, permissions, corrections, and evidence.
AI models are replaceable executors operating inside that control plane.

Provider-specific behavior must be isolated behind model adapters. Notion
operating DNA, Git specifications and tests, Runtime Events and correction
history, and the Tool Gateway remain independent from the selected model.

Model succession follows this order:

1. review participation;
2. shadow mode;
3. bounded domain succession;
4. primary-role transition.

Every stage retains a rollback path. Pulda does not perform an immediate,
unverified full-provider replacement.

## Verification

A candidate model is evaluated with Pulda-specific scenarios covering:

- authoritative-source retrieval;
- preservation of human facts and corrections;
- reuse of a correction in the next comparable interpretation;
- tool calls and permission boundaries;
- cost, latency, recovery, and data control;
- reproducible handoff without provider-only memory.

## Consequences

- Approved decisions must be externalized in provider-independent formats.
- Model-specific prompts or capabilities cannot become the sole operating
  memory of Pulda OS.
- Provider transition requires comparative evidence and human approval.
- Runtime implementation remains pending and requires a separate approved CR,
  acceptance criteria, tests, and execution evidence.
