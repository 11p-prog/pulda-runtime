## Original content (verbatim, renumbered only)

### Problem
The current center-tab structure exposes Event, Task, Goal, and Project as
equal navigation concepts. The event list lacks a visible captured-time
axis, making repeated entries feel like unstructured storage.

### Requested change
1. Remove central Event/Task/Goal/Project tabs from Home.
2. Make Activity Feed the sole central Home surface.
3. Add captured-time display to every event row.
4. Group feed by date periods.
5. Keep newest-first ordering.
6. Add an AI Insight side panel or responsive secondary section.
7. Preserve all original event records.

### Data requirements
- Ensure every event has captured_at.
- Backfill captured_at from created_at where available.
- Do not conflate captured_at, occurred_at, and scheduled_at.

### Acceptance criteria
- AC1: Home opens directly to event capture and feed.
- AC2: No central entity-type tabs remain.
- AC3: Every event shows captured time.
- AC4: New events appear immediately at the top.
- AC5: Date grouping works across today/yesterday/older records.
- AC6: Existing event records remain intact.
- AC7: Mobile layout places AI Insight below the feed.
