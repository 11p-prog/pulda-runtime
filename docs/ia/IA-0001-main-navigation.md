# IA-0001 Main Navigation

Status: Proposed for MVP

## Decision
Remove the central data-model tabs that expose Event / Task / Goal / Project as equal concepts.

## Main navigation
- Home
- Projects
- Routine
- Knowledge
- Archive
- Settings

## Home center area
Home has one primary surface: Activity Feed.

## Secondary concepts
- Event: capture unit; visible in feed.
- Task: actionable interpretation of an event; not a top-level tab.
- Project: managed workspace, available from navigation.
- Goal: contextual target inside projects/areas, not a permanent Home tab.

## Rule
Users navigate by purpose, not by database entity type.
