# UX-0003 AI Insight

## Purpose
Show useful structure discovered from events without forcing the user to classify each event.

## Home panel contents
- Repeated contexts
- Recently increasing activity
- Stalled patterns
- Candidate project suggestions

## Behavior
- AI proposes; user approves or dismisses.
- No automatic destructive regrouping.
- Suggestions must cite the related event count and time range.

## Example
Pulda Runtime activity
- 7 related events in 3 days
- Last activity: 16:32
- Action: Review as project candidate
